## ðŸš€ Langkah Instalasi

1. **Clone repositori**
   ```bash
   git clone https://github.com/TheMordhon1/parking-system-fe.git
   cd parking-system-fe
   ```
2. **Instal dependensi**
   ```bash
   npm install
   ```
3. **Jalankan Aplikasi**
   ```bash
   npm run dev
   ```
4. **(Optional) Inisialisasi shadcn/ui (jika baru pertama kali setup)**

   ```bash
   npx shadcn@latest init
   ```

5. **Tambahkan komponen shadcn yang diperlukan**
   ```bash
   npx shadcn@latest add [nama-komponen]
   ```
